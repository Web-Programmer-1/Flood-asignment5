What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node in the tree corresponds to a part of the document (e.g., elements, attributes, and text content). This allows developers to interact with, manipulate, and update the document's structure, style, and content dynamically via programming languages, most commonly JavaScript.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events for its child elements, rather than attaching individual listeners to each child. This works because events in the DOM bubble up (or propagate) from the target element (where the event occurred) to its parent and then up through ancestors until they reach the root of the document.
